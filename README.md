# homenetwork

## Clone the Repository and run the installer

```bash
sudo apt install git -y
git clone https://github.com/nicopret/homenetwork.git
```

## monitoring

go to `raspi-monitor` and run:

`sudo chmod +x setup*`

then

- setup_prometheus.sh
- setup_node_exporter.sh
- setup_grafana.sh
- setup_nginx.sh
